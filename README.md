# LiveMusicStPete

Find Local Live Music Venues in St. Petersburg using CRUD based API's. Users can add, edit, update and delete from the database as they please.

-C#, SQL, ASP.NET, REST API, EF Core, LINQ, Typescript, React, HTML, CSS
